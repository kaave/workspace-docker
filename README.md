## How to use
### create/download image

```bash
$ /usr/bin/env bash ./setup.sh
```

### start on Daemon mode

```bash
$ docker-compose up -d
```

### access to workspace:dev on ssh

```bash
$ ssh USERNAME@localhost -p 2222
# use sshpass
$ sshpass -p USERPASS ssh USERNAME@localhost -p 2222
```
